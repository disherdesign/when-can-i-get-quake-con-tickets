//DESCRIPTION
this program should send a email if a change in the /registration-2 page of 
quakecon.org is detected.

changes in the registration page are set to be checked once every three hours
after the program is launched.

set your email, password, and smtp server up in the email.go source file
in the sendNotificationEmail() function.

//INSTALLATION
run or build main.go and email.go together.
